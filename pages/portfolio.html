EcoMarket

O EcoMarket é uma plataforma de comércio eletrônico que se concentra em conectar consumidores a produtos sustentáveis e eco-friendly. Os usuários podem encontrar uma variedade de produtos orgânicos, reciclados e sustentáveis, desde alimentos e produtos de beleza até moda e artigos para casa.

Tecnologias:

HTML5 CSS3 Angular.js MongoDB
Github

Safe Neighborhood

O Safe Neighborhood é um aplicativo mobile de segurança que permite aos usuários consultar dados de possíveis ocorrências e relatos, levando em consideração sua localização geográfica. O aplicativo apresenta um mapa interativo que identifica zonas mais seguras e perigosas, juntamente com uma lista de ocorrências recentes e estatísticas detalhadas sobre lugares e horários com maior e menor índice de segurança.

Tecnologias:

Flutter Dart
Github

TravelBuddy

O TravelBuddy é um aplicativo de planejamento de viagens que ajuda os viajantes a organizar e explorar novos destinos. Os usuários podem pesquisar e reservar voos, hotéis e atividades, criar itinerários personalizados e compartilhar dicas e recomendações com outros viajantes.

Tecnologias:

HTML5 CSS3 React.js MySQL
Github
